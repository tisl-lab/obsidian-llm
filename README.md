# obsidian-llm

Ce template obsidian est conçu pour offrir des outils aux élevés de l'ETS comme outils de gestion d'études et de prise de notes. Il comprend une multitude d'outils qui peuvent être utilise a travers le parcoure scolaire.

Les outils inclus sont séparateur en 2 catégories principales:

**Outils de gestion :** Ces outils sont présent pour aider a la gestion des études, gestion des remises et tout ce qui est lie a la gestion des études.
Une présentation de chaque outils est présent dans [[vault/Outils de gestion]].

**Outils d'études :** Ces outils sont présent pour aider dans les études tel que la révision, la gestion de notes et tout ce qui est lie a l'étude.
Une présentation de chaque outils est présent dans [[vault/Outils d'etude]].

## Pourquoi obsidian

Si vous utilisez déjà notion, vous vous demandez surement pourquoi utiliser obsidian. Voici les avantages qu'offre obsidian par rapport a notion:

1. **Disponibilité offline:** Notion nécessite une connexion internet pour fonctionner, obsidian fonctionne sans connexion internet.
2. **Personnalisation**: Obsidian offre une multitude de plugins et de thèmes pour personnaliser l'application de l'interface aux fonctionnalité.
3. **Ouvert aux développeurs**: Malgré le fait que obsidian n'est pas open source, il offre une API pour les développeurs qui veulent créer des plugins pour étendre les fonctionnalités de l'application.
4. **Format de notes**: Alors que notion utilise un format de page propriétaire, obsidian utilise des fichiers markdown qui sont facilement exportable et lisible par n'importe quel éditeur de texte. Ceci est probablement le plus gros avantage de l'application, vous ne vous en rendez peut-être pas compte mais les entreprises propriétaires peuvent décider d'arrêter d'offrir un service du jour au lendemain et vous perdrez toutes vos notes. Cette situation peux sembler improbable mais elle est déjà arrive a plusieurs reprises (Evernote, Google Inbox, Microsoft Groove Music). Avec obsidian, toutes les notes sont stocker sur votre appareil et vous avez la garanti de pouvoir retrouver vos notes peux importe le futur de l'application et ce pour l'éternité.
